Pure python models
------------------
GPSat model based on a pure python implementation of GP regression.
Based on `this code <https://github.com/William-gregory/OptimalInterpolation>`_ to replicate the
optimal interpolation algorithm in [G'21].

[G'21] Gregory, William, Isobel R. Lawrence, and Michel Tsamados. "A Bayesian approach towards daily pan-Arctic sea ice freeboard estimates from combined CryoSat-2 and Sentinel-3 satellite observations." The Cryosphere, 2021.
